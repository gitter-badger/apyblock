# ashblock

A Shell Blocker, the script for blocking ads on the web.

# Installation

- Check if `wget` is installed.

- Just copy `ashblock` to your `$PATH`.

An build script for RoseSpine is coming.

# Usage

```
sudo ashblock update
```
